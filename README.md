# Lane Detection Project

This project implements a real-time lane detection system using Python, OpenCV and NumPy.

## Features
- Detects left and right lane lines
- Uses Canny edge detection
- Uses Hough Transform
- Applies smoothing for more stable lane detection
- Calculates lane offset

## Technologies
- Python
- OpenCV
- NumPy

## Files
- `lane_detection_pro.py` - main source code
- `road.mp4` - input video
- `Lane_Detection_Report.docx` - project report

## How to run
```bash
pip install -r requirements.txt
python lane_detection_pro.py
